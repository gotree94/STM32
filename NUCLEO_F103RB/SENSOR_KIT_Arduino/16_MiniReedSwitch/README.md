# 미니 리드 스위치 모듈 (Mini Reed Switch Module)

NUCLEO-F103RB 보드를 이용한 미니 리드 스위치 모듈 테스트 프로젝트입니다.

## 📋 개요

리드 스위치는 자기장에 반응하여 접점이 열리거나 닫히는 스위치입니다. 자석이 가까이 오면 내부의 리드가 접촉하여 회로가 연결됩니다. 도어/창문 개폐 감지, 보안 시스템 등에 활용됩니다.

## 🔧 하드웨어 구성

### 필요 부품
| 부품 | 수량 |
|------|------|
| NUCLEO-F103RB | 1 |
| 미니 리드 스위치 모듈 | 1 |
| 점퍼 와이어 | 3 |

### 핀 연결

| 모듈 핀 | NUCLEO 핀 | 설명 |
|---------|-----------|------|
| VCC | 3.3V / 5V | 전원 |
| GND | GND | 접지 |
| DO | PA0 (A0) | 디지털 출력 |

## 💻 소프트웨어 설정

### Arduino IDE 설정
1. **보드 매니저**: STM32 MCU based boards 설치
2. **보드 선택**: Nucleo-64 → Nucleo F103RB
3. **시리얼 포트**: 해당 COM 포트 선택

### 라이브러리
- 기본 라이브러리만 사용 (추가 설치 불필요)

## 📝 동작 설명

1. 자석이 리드 스위치에 가까워지면 내부 접점이 닫힘
2. DO 핀이 LOW 상태가 되어 자석 감지
3. 내장 LED가 켜지고 시리얼 모니터에 상태 출력
4. 자석이 멀어지면 접점이 열리고 HIGH 상태로 복귀

## 🖥️ 시리얼 출력 예시

```
================================
미니 리드 스위치 모듈 테스트
NUCLEO-F103RB
================================
자석을 가까이 가져가면 상태가 변합니다.

[대기] 자석이 감지되지 않습니다. (스위치 OFF)
[감지] 자석이 감지되었습니다! (스위치 ON)
[대기] 자석이 감지되지 않습니다. (스위치 OFF)
```

## ⚠️ 주의사항

- 리드 스위치는 물리적 접점 소자로 디바운싱 처리 필요
- 강한 충격에 취약하므로 주의하여 취급
- 자석의 극성과 거리에 따라 감도가 달라질 수 있음
