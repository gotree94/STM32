# 레이저 모듈 (KY-008 Laser Module)

NUCLEO-F103RB 보드를 이용한 레이저 모듈 테스트 프로젝트입니다.

## 📋 개요

KY-008 레이저 모듈은 650nm 파장의 빨간색 레이저를 출력하는 모듈입니다. 간단한 디지털 제어로 ON/OFF가 가능하며, PWM을 이용한 밝기 조절도 가능합니다.

## 💻 소프트웨어 설정

### Arduino IDE 설정
1. **보드 매니저**: STM32 MCU based boards 설치
2. **보드 선택**: Nucleo-64 → Nucleo F103RB
3. **시리얼 포트**: 해당 COM 포트 선택

### 라이브러리
- 기본 라이브러리만 사용 (추가 설치 불필요)

## 📝 동작 모드

시리얼 모니터에서 1-5 숫자를 입력하여 모드 선택:

| 명령 | 모드 | 설명 |
|------|------|------|
| 1 | ON | 레이저 켜기 |
| 2 | OFF | 레이저 끄기 |
| 3 | BLINK | 500ms 간격 깜빡임 |
| 4 | SOS | 모스부호 SOS 패턴 |
| 5 | FADE | PWM 밝기 조절 |

### SOS 패턴
```
S: ● ● ●     (짧게 3번)
O: ━ ━ ━     (길게 3번)  
S: ● ● ●     (짧게 3번)
```

## 📊 모듈 사양

| 항목 | 값 |
|------|-----|
| 동작 전압 | 5V |
| 파장 | 650nm (빨간색) |
| 출력 | 5mW (Class 3R) |
| 동작 전류 | ~30mA |

## 📚 응용 분야

- 레이저 정렬/포인터
- 물체 감지 (수광 센서와 조합)
- 거리 측정 (ToF 방식과 조합)
- 레이저 통신
- 레이저 트립와이어 (보안)

## 🔧 레이저 + 수광 센서 조합 예시

레이저와 포토레지스터를 조합하여 레이저 빔 차단 감지 시스템을 만들 수 있습니다.

```
레이저 ))) → → → → ((( 포토레지스터
          물체가 지나가면 빔 차단 감지
```

## ⚠️ 주의사항

1. **눈 보호**: 절대 직접 응시 금지
2. **반사 주의**: 거울이나 반사 물체 주의
3. **연속 사용**: 장시간 사용 시 발열 가능
4. **전원**: 5V 권장 (3.3V에서는 밝기 감소)

## 🔍 트러블슈팅

| 증상 | 해결 방법 |
|------|----------|
| 레이저가 켜지지 않음 | 배선 확인, 5V 전원 확인 |
| 밝기가 약함 | 5V 전원 사용, 모듈 손상 확인 |
| PWM 동작 안 함 | PWM 지원 핀 사용 확인 |
