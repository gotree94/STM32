# 차량 시뮬레이션 ICD (Interface Control Document)

## 1. 통신 사양

| 항목 | 값 |
|------|-----|
| Baud Rate | 115200 |
| Data Bits | 8 |
| Parity | None |
| Stop Bits | 1 |
| Flow Control | None |
| 물리 계층 | RS-232 (USB-Serial) |

## 2. 패킷 구조

### 2.1 형식

```
{{%07.3f,,%07.3f,,%07.3f,,%07.3f,,%07.3f}}
```

- 시작: `{{` (2 bytes)
- 데이터 필드: 5개, `,,` (2 comma) 로 구분
- 종료: `}}` (2 bytes)
- 전체 패킷 크기: **47 bytes**

### 2.2 필드 상세

| 필드 | 인덱스 | 형식 | 범위 | 설명 |
|------|--------|------|------|------|
| LEFT_PWM | 1 | `%07.3f` | 0.000 ~ 999.999 | 왼쪽 바퀴 PWM 값 (소수점 이하 무시, 정수 0~999) |
| RIGHT_PWM | 2 | `%07.3f` | 0.000 ~ 999.999 | 오른쪽 바퀴 PWM 값 (소수점 이하 무시, 정수 0~999) |
| FLAGS | 3 | `%07.3f` | 0.000 ~ 999.999 | 제어 플래그 (소수점 이하 무시, 정수 0~999) |
| SERVO1 | 4 | `%07.3f` | 0.000 ~ 999.999 | 서보모터 1 각도 (0~180도 매핑, 기본값 90) |
| SERVO2 | 5 | `%07.3f` | 0.000 ~ 999.999 | 서보모터 2 각도 (0~180도 매핑, 기본값 90) |

### 2.3 FLAGS 필드 상세

FLAGS 정수값의 각 자리수는 비트 필드로 동작:

```
x-- : 레이저 포인터 점등 여부 (hundreds)
-x- : 오른쪽 구동모터 출력방향 (tens)
--x : 왼쪽 구동모터 출력방향 (units)
```

| 조건 | 값 |
|------|-----|
| 해당 자릿수 > 5 | ON (True) |
| 해당 자릿수 <= 5 | OFF (False) |

예시:
- `005` → units=5 (≤5), tens=0 (≤5), hundreds=0 (≤5) → 전부 OFF
- `567` → units=7 (>5), tens=6 (>5), hundreds=5 (≤5) → 좌/우 모터 ON, 레이저 OFF
- `789` → units=9 (>5), tens=8 (>5), hundreds=7 (>5) → 전부 ON

## 3. 각도 매핑

SERVO1 / SERVO2 값은 "0 ~ 999" 범위로 수신되며, 물리 각도 "0° ~ 180°"로 선형 매핑:

```
물리각도 = round(수신값 × 180 / 999)
```

| 수신값 | 물리 각도 |
|--------|----------|
| 0 | 0° |
| 500 | 90° (기본값) |
| 999 | 180° |

## 4. 예시 패킷

### 전진
```
{{500.000,,500.000,,005.000,,500.000,,500.000}}
```
- LEFT_PWM=500, RIGHT_PWM=500
- FLAGS=005 → 모터 방향 OFF, 레이저 OFF
- SERVO1=90°, SERVO2=90°

### 정지
```
{{000.000,,000.000,,005.000,,200.000,,700.000}}
```
- LEFT_PWM=0, RIGHT_PWM=0
- FLAGS=005 → 모터 방향 OFF, 레이저 OFF
- SERVO1≈36°, SERVO2≈126°

### 전체 ON
```
{{999.000,,999.000,,789.000,,000.000,,999.000}}
```
- LEFT_PWM=999, RIGHT_PWM=999
- FLAGS=789 → 좌/우 모터 방향 ON, 레이저 ON
- SERVO1=0°, SERVO2=180°

## 5. 색상 매핑 (표시용)

수신된 PWM 값 (0~999) 은 GUI에서 레인보우 색상으로 표시:

| PWM 값 | Hue | 색상 |
|--------|-----|------|
| 0 | 0° | 빨강 |
| ~166 | 50° | 주황/노랑 |
| ~333 | 100° | 연두 |
| ~500 | 150° | 청록 |
| ~666 | 200° | 하늘/파랑 |
| ~833 | 250° | 보라 |
| 999 | 300° | 자주 |
